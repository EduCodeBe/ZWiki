This zope product allows you to build wiki webs in zope. To install:

1. unpack ZWiki-x.x.x.tgz in your zope Products directory

2. restart zope

3. in the ZMI, add a Wiki

For documentation and assistance, please see http://zwiki.org, or post on
http://zwiki.org/GeneralDiscussion. All feedback, bug reports and other
help appreciated.
